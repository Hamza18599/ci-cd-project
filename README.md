# Hamza CI/CD + GitOps Project

This project includes:

- CI/CD with GitHub Actions
- ArgoCD GitOps deployment
- Kustomize overlays
- Argo Rollouts canary deployment
- Automatic image updating
- Demo Node.js app
- Ready for Lens + your cluster

## Run locally
```
npm install
npm start
```

## Build Docker
```
docker build -t myapp .
```
